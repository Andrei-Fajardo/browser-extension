# Mouse Parts Lookup

Browser extension for mouse repair technicians to look up **sourced, verifiable** mouse parts (switches, encoders, sensors, MCUs) by brand/model—without hallucinated specs.

## Status

Early development. Product requirements are in [`docs/PRD.md`](./docs/PRD.md).

## Principles

- **Accuracy over completeness** — unknown fields stay blank.
- **Provenance required** — every filled field cites a source.
- **Scraping for gaps** — allowlisted pipelines gather data; nothing is invented.
